# Official logo overrides

Put your local official PNG files here.

File names must match the generated logo file names from:

```text
public/logos/college
```

Example:

```text
public/logos/college-official/alabama.png
public/logos/college-official/ohio-state.png
public/logos/college-official/notre-dame.png
```

The game checks this folder first.

If a file exists, it is shown.
If it does not exist, the game falls back to:

```text
public/logos/college
```
