# Friday Night Dynasty — Architecture

## Цель игры

Friday Night Dynasty — браузерная игра про живой мир американского футбола.

Главный фокус:

- школьный футбол;
- колледжи;
- рекрутинг;
- поколения игроков;
- тренерские династии;
- история мира;
- симуляция сезонов.

## Главный закон проекта

Сначала живой мир. Потом интерфейс. Потом красота.

## Структура проекта

src/
core/
app/
storage/
content/

## Core

src/core содержит всю игровую логику:

- генерация мира;
- генерация игроков;
- генерация команд;
- календарь;
- симуляция матчей;
- симуляция недель;
- завершение сезона;
- история мира;
- рейтинги;
- рекрутинг.

Core не импортирует React.

## App

src/app содержит интерфейс:

- экраны;
- компоненты;
- Zustand store;
- стили.

App вызывает функции core, но не хранит правила мира внутри компонентов.

## Storage

src/storage отвечает за сохранения:

- IndexedDB;
- Dexie;
- миграции сейвов;
- загрузка мира;
- удаление сейвов.

## Content

src/content содержит данные:

- имена;
- названия школ;
- названия городов;
- шаблоны новостей;
- черты игроков;
- события.

## Правило Pull Request

Каждая новая функция делается в отдельной ветке и попадает в main только через Pull Request.
