# Friday Night Dynasty v1.2.2 — UI Pagination + Rating Scale

## Команда проекта

```bash
cd "C:\FridayNightDynasty\friday_night_dynasty_v01"
```

## Что добавлено

### 1. Команды стали тонким списком

Во вкладке `Команды`:

- карточки заменены тонкими строками;
- команды сортируются по OVR от сильнейшей к слабейшей;
- рядом с названием вкладки показывается общее количество команд;
- OVR выделен жирным белым цветом;
- список разбит на страницы.

### 2. Таблица колледжей свёрнута по дивизионам

Во вкладке `Таблица` для колледжей:

- каждый дивизион/конференция сначала свёрнут;
- игрок сам нажимает `Развернуть`.

### 3. Убрана История лиги

Из вкладки `История` убран блок `История лиги`.

### 4. Добавлено 3000 имён и фамилий

Файл:

```text
src/content/names.ts
```

Теперь содержит:

```text
1500 first names
1500 last names
```

Смесь US + Mexican/Latino.

### 5. Неделя перенесена в Управление

Теперь текущий год/неделя показывается в блоке:

```text
Управление
```

Белым жирным текстом.

### 6. OVR команд завязан на состав

Школьные команды используют расчёт OVR от состава.
Колледжские команды в списке используют `getCollegeRosterStrength`.

### 7. Дерби 1-3 на команду

Школы и колледжи теперь получают 1-3 rivalry-связи.

### 8. Переключатель переименован

Было:

```text
Уровень I
Уровень II
```

Стало:

```text
Школа
Колледж
```

### 9. Ростеры стали страницами

Пагинация добавлена для:

- ростера школьной команды;
- ростера колледжской команды;
- коммитов;
- проспектов.

### 10. Сортировка ростеров по заголовкам

В ростерах можно нажимать на:

```text
поз
класс
общ
пот
возраст
```

и сортировать список.

### 11. Новая шкала OVR

Игроки:

```text
школа: 0-40
колледж: 30-70
NFL: 60-100 позже
```

При переходе из школы в колледж:

- OVR поднимается в колледжский диапазон;
- potential пересчитывается заново от школьного потенциала + случайность.

## Важно про логотипы

Официальные trademark-логотипы ESPN не включены в zip.

В игре уже есть `logoAsset`, а PNG-бейджи лежат тут:

```text
public/logos/college
```

Их можно заменить официальными файлами с теми же именами, если ты используешь их локально.
