## Что исправлено

Этот патч чинит главные проблемы v1.0:

- колледжи больше не являются пустым списком без карточек;
- у колледжей есть полноценный профиль команды;
- в профиле колледжа есть вкладки:
  - обзор;
  - состав;
  - календарь;
  - история;
- из списка колледжей можно открыть профиль колледжа;
- из таблицы колледжей можно открыть профиль колледжа;
- из календаря колледжей можно открыть команды;
- у колледжей появились rivalry-пары;
- в календаре колледжей дерби отмечаются как `дерби`;
- колледжский сезон можно переключить отдельно от старшей школы;
- после завершения колледжского сезона появляется кнопка `Новый сезон колледжей`;
- игрок, который уже перешёл в колледж, больше не открывается как `выпускник школы`;
- если открыть старую карточку выпускника с коммитом, игра перекидывает на текущего college player;
- избранное корректно показывает converted players как игроков колледжа;
- история колледжских команд записывается для всех команд, а не только для финалистов.

## Важно по старым сохранениям

Патч умеет нормализовать старые миры:

- добавляет `rivalryIds` старым колледжам;
- добавляет отсутствующие college-season поля;
- переводит старые commitment-записи к новой модели.

Если видишь старые кривые карточки из сохранения, лучший тест:

```text
Новый мир → симулировать школьный сезон → перейти к новому сезону → колледжи
```
