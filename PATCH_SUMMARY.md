# Friday Night Dynasty v1.0.3 — Unified World Season

## Команда проекта

```bash
cd "C:\FridayNightDynasty\friday_night_dynasty_v01"
```

## Главная правка

Удалена неправильная логика, где школьный сезон и колледжский сезон могли жить в разных годах.

Теперь есть один год мира:

```text
2027 = школьный сезон 2027 + колледжский сезон 2027
```

Переход года общий:

```text
школьный сезон завершён
+ колледжский сезон завершён
= можно перейти в 2028
```

## Что изменено

### 1. Единая симуляция недели

Кнопка теперь симулирует неделю мира:

```text
Симулировать неделю мира
```

Она двигает:

- школьный сезон, если он ещё не завершён;
- колледжский сезон, если он ещё не завершён.

### 2. Единая симуляция года

Кнопка:

```text
Симулировать год мира
```

завершает оба уровня в одном и том же году:

- школу;
- колледжи.

### 3. Один переход года

Удалена отдельная логика:

```text
Новый сезон колледжей
```

Теперь только:

```text
Перейти к новому году
```

Она доступна только когда завершены оба сезона.

### 4. Старые сохранения с кривыми годами чинятся

Если в сохранении было:

```text
школа 2027
колледжи 2031
```

normalize сбрасывает college season обратно к году мира:

```text
школа 2027
колледжи 2027
```

При таком ремонте сбрасывается только кривой текущий college season, чтобы не тащить будущие матчи в прошлый год.

### 5. Отключён отдельный advanceCollegeSeason

Файл оставлен как совместимая заглушка, чтобы старые импорты не ломали build, но фактически отдельный переход колледжского года больше не используется.
